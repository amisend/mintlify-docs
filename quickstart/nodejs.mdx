## Installation

<CodeGroup>
  ```bash npm
npm install --save amisend
```

```bash yarn
yarn add amisend
```
</CodeGroup>

### Authentication

Create an account at [amisend](https://amisend.com) and get your **API Key**.

```js
// CommonJS
const Amisend = require("amisend");

// ES6 / Typescript
import Amisend from "amisend";

const amisend = new Amisend({
  apiKey: "YOUR_API_KEY",
});
```
