
```bash yarn
yarn add amisend
```

## Getting Started

### Authentication

Create an account at [Amisend](https://amisend.com) and get your [API Key](https://amisend.com/api-keys).

```js
// CommonJS
const Amisend = require("amisend");

// ES6 / Typescript
import Amisend from "amisend";

const amisend = new Amisend({
  apiKey: "YOUR_API_KEY",
});
```

## Emails

### Send an email

```js
amisend.emails
  .send({
    from: "Example <example@domain.com>",
    to: ["example@hotmail.com"],
    subject: "hello world",
    html: "<p>It works!</p>",
  })
  .then((response) => {
    // Handle response
  })
  .catch((error) => console.log(error)); // Handle error
```
